  1. Instalar un servidor SSH en Linux.

Para empezar, descargamos la herramienta de servidor de SSH con el comando sudo apt install openssh-server.

Luego, usando el comando sudo adduser <nombre_de_usuario>, añadimos el usuario practica en nuestro caso y rellenamos los pertinentes campos y veremos con ls /home que se ha creado dicho usuario.

Siguiendo con el apartado, instalamos Putty en nuestra máquina con Windows 10 y ponemos la dirección IP y el puerto para conectarnos a nuestra máquina ubuntu server y podemos ver que nos permite la conexión.

Ahora, probamos que tenemos conexión entre una máquina Ubuntu normal y nuestra máquina con Ubuntu server, que podemos ver que sí que nos deja efectuar la conexión.

¿Cómo se podría conectar un usuario sin tener que meter la contraseña cada vez que se conecta?

Primero de todo, creamos en la o las máquinas clientes un par de claves ssh y luego copiamos la o las claves públicas en nuestro servidor en que tengamos instalado el servicio ssh con el comando ssh-copy-id usuario@ip, reemplazando usuario por el nombre de usuario deseado y ip por la dirección ip de nuestro servidor y luego, para terminar ya deberíamos de poder conectarnos sin tener que ingresar la contraseña, solo el usuario y la dirección IP del servidor.

  1. Conexión de escritorio remoto en Windows (RDP)

Para empezar, en la máquina de Windows Server vamos a la administración del servidor y en el apartado de servidor local, activamos la opción de escritorio remoto y reiniciamos el servidor.

Ahora, deberemos de conectarnos con una máquina windows, que en nuestro caso es una máquina con windows 10 home y una máquina de ubuntu, en la cual deberemos de instalar remmina para conectaros.

Podemos ver que efectivamente, nos deja conectarnos mediante la máquina de windows.

En nuestra máquina con ubuntu, instalamos remmina. Después de instalar la aplicación, abrimos la aplicación de remmina que acabamos de instalar e introducimos la dirección IP de nuestro servidor con Windows.

Después de este paso, deberemos de ingresar el nombre de usuario, la contraseña y si se tiene, el dominio y le damos a conectar y podemos ver que tenemos conexión.

  1. Instalar un Teamviewer en equipo windows y en otro Linux.

Para empezar, en esta práctica vamos a conectarnos a nuestra máquina de windows 11 desde ubuntu y a un equipo ubuntu desde windows 11 y a ambas desde un móvil. Primero de todo, deberemos de descargar teamviewer en la versión de host o en la versión completa.

Desde Ubuntu, primero descargamos la herramienta y en windows más de lo mismo, y entonces nos conectamos desde Ubuntu a nuestra máquina de Windows 11 y vemos que podemos controlarla.

Ahora nos conectaremos desde la máquina de windows a la de ubuntu, y podemos ver que sí que tenemos conexión.

Por último, nos conectaremos desde un dispositivo móvil, en la que podemos observar que la interfaz es diferente, pero que podemos controlar el ordenador.

Ahora nos centraremos en conectarnos en modo de presentación, de la misma manera de Windows 11 a bunu y viceversa y desde el móvil. Para ello, iremos al apartado de Reunion y seleccionamos la opción de compartir pantalla y veremos que podemos compartir nuestra pantalla desde cualquier dispositivo.
