Proyecto ChefBot

Guía de implementación de un LLM especializado (Fine-Tuning Local)
Fase 1

Preparación del Entorno

NVIDIA RTX 5060 PyTorch Nightly CUDA 13

El primer paso crítico es validar la capacidad de cómputo. Se verifica que la GPU dedicada está disponible y se instalan las dependencias compatibles con la arquitectura más reciente.

Análisis: La captura muestra la instalación de librerías y la monitorización de recursos, confirmando el uso de una GPU Laptop RTX 5060 lista para la carga de trabajo.
Fase 2

Estructura del Proyecto

VS Code venv Python Scripts

Organización del espacio de trabajo. Se separan claramente los scripts de entrenamiento (train), fusión (merge) y los datos, utilizando un entorno virtual aislado.

Fase 3

Curación del Dataset

JSONL Instruction-Tuning

Definición de la "personalidad" del modelo. Se utiliza un archivo JSONL con pares de Instrucción (ingredientes) y Output (receta) para especializar al modelo base.

Fase 4

Configuración del Modelo

Phi-3-mini QLoRA (4-bit) BitsAndBytes

Configuración del script Python. Se utiliza la técnica QLoRA para cargar el modelo en 4 bits, permitiendo el entrenamiento en hardware de consumo con memoria limitada.

Fase 5

Ejecución del Entrenamiento

Training Loop Loss Monitoring

El proceso de aprendizaje en acción. Las barras de progreso muestran cómo el modelo itera sobre el dataset, reduciendo el error (loss) progresivamente.

Fase 6

Resultados: Adaptadores LoRA

Safetensors Adapter Config

Al finalizar, obtenemos los "adaptadores" en la carpeta de salida. Estos archivos ligeros contienen el nuevo conocimiento culinario adquirido.

Fase 7

Merging (Fusión)

PeftModel Merge & Unload

Script para consolidar el modelo. Se carga el modelo base y los adaptadores LoRA para fusionarlos en un único modelo independiente listo para usar.

Fase 8

Prueba Final y Despliegue

Inferencia Chatbot

El resultado final. ChefBot está activo y responde coherentemente a preguntas de cocina, demostrando que el fine-tuning ha sido exitoso.

Verificación: Como indicaste, esta es la parte final donde cargamos el chatbot y verificamos su funcionamiento real interactuando con él.
